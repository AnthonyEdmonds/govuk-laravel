# H1

Create a first-level heading. 

```html
<x-govuk::h1
    id="my-id"
    size="l"   
    marginBottom="null"
>
    My Heading
</x-govuk::h1>
```

## Props

| Name         | Type    | Default | Description                 |
|--------------|---------|---------|-----------------------------|
| id           | string  | null    | The ID of the heading       |
| size         | string  | l       | How large the heading should be |
| marginBottom | integer | null    | How much the margin should be|
